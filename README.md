# 👋 Olá!

Sou **Herick José de Freitas**, estudante de **Ciência da Computação (6º período)** com interesse em **desenvolvimento de software, sistemas distribuídos e inteligência artificial aplicada**.

Atualmente atuo como **pesquisador no ARIA/UFPB**, contribuindo para o desenvolvimento de sistemas utilizados pela **SEFAZ-PB** dentro do projeto **CACTO**. Tenho experiência com **desenvolvimento fullstack, APIs em Python e sistemas de processamento de dados**.

Também sou **diretor executivo da *Solstício Estúdios***, uma startup focada no desenvolvimento de jogos digitais independentes.

Sou entusiasta de **Machine Learning e Visão Computacional**, área na qual venho desenvolvendo projetos utilizando **Python, OpenCV e modelos de detecção de objetos**.
